# go-glamour-wrapper
A very simple CLI wrapper for the [charmbracelet/glamour](https://github.com/charmbracelet/glamour).

## Installation
1. Clone the project
2. Run `make`
3. You can run the program with `catmd EXAMPLE.md`
4. Ready to roll ЁЯЪА

тЪая╕П  The installer moves the executable into `/usr/local/bin`. Update the `Makefile` if you want to change the installation path!
